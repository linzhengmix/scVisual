# Custom x-axis for grid plots

Create a custom x-axis for grid plots

## Usage

``` r
grid.xaxis2(
  at = NULL,
  breaks = 5,
  labels = NULL,
  tick.len = 0.5,
  label.space = 0.5,
  side = c("bottom", "top"),
  rot = 0,
  label.size = 12
)
```

## Arguments

- at:

  Position of ticks

- breaks:

  Number of breaks

- labels:

  Tick labels

- tick.len:

  Tick length

- label.space:

  Space between ticks and labels

- side:

  Side of the axis (bottom or top)

- rot:

  Label rotation

- label.size:

  Label font size

## Value

None
