# Custom color key for grid plots

Create a custom color key for grid plots

## Usage

``` r
grid.colorkey(
  x = NULL,
  color = NULL,
  color.n = 100,
  ticks.side = c("left", "right", "top", "bottom"),
  pos = c("h", "v")
)
```

## Arguments

- x:

  Data values for color scaling

- color:

  Color palette

- color.n:

  Number of color levels

- ticks.side:

  Side for ticks

- pos:

  Orientation (horizontal or vertical)

## Value

None
