# using cellRatioPlot to visualize cell percent ratio across clusters

using cellRatioPlot to visualize cell percent ratio across clusters

## Arguments

- object:

  object seurat object.

- sample.name:

  sample name in meta info, default NULL.

- celltype.name:

  celltype name in meta info, default NULL.

- col.width:

  column width, default 0.7.

- flow.alpha:

  flow color alpha, default 0.25.

- flow.curve:

  flow curve rate, default 0.

- fill.col:

  supply your own fill colors, default NULL.

## Value

a ggplot object.
