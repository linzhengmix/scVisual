# using drawLegend to add cluster id in legend

using drawLegend to add cluster id in legend

## Arguments

- object:

  object seurat object, default NULL.

- plot:

  ggplot object, default NULL.

- cell_type:

  cell type column name, default NULL.

- clusters:

  cluster id column name, default NULL.

- ncol:

  ncols to draw legend(1/2), default 1.

- col:

  point color, default hue_pal().

- pt.size:

  point size, default 8.

- text.size:

  text size, default 4.

## Value

combine plot
